# 封面图片说明

本文需要以下图片文件：

## 必需图片
1. `cover.jpg` - 文章封面图片
   - 建议尺寸：1200×630像素

## 图片建议
- 使用高质量的图片，确保清晰度
- 图片大小适中，避免过大影响加载速度
- 可以使用绘图工具（如draw.io、PPT等）创建示意图
- 可以使用实际训练结果截图作为示例图片
