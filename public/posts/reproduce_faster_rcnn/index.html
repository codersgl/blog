复现 Faster R-CNN：从理论到实践

引言

在深度学习的学习过程中，我逐渐意识到理论知识与工程实践之间存在显著差距。为了弥合这一差距，我决定通过复现经典算法来深入理解深度学习项目的完整流程。相比于仅仅阅读论文或使用现成的库，亲手实现算法能够让我更好地掌握实现细节、调试技巧和优化策略。

在众多深度学习算法中，我选择了目标检测领域的里程碑式工作——Faster R-CNN。这个算法不仅具有重要的理论价值，而且在工业界有着广泛的应用。相比于基础的图像分类网络，Faster R-CNN涉及更复杂的多任务学习；而相比于后来的单阶段检测器（如YOLO系列），它的架构又相对清晰，适合作为学习目标检测的入门项目。

本文详细记录了我从零开始复现Faster R-CNN的完整过程，涵盖了数据准备、网络架构设计、损失函数实现、训练策略以及实验结果分析。我希望通过这篇文章，不仅总结自己的学习经验，也能为其他想要深入理解目标检测算法的同学提供参考。

Faster R-CNN 算法原理

算法概述

Faster R-CNN（Region-based Convolutional Neural Networks）是目标检测领域的一个重要里程碑。由任少卿等人于2015年提出，该算法通过引入区域提议网络（Region Proposal Network, RPN），将目标检测任务统一到一个端到端的深度学习框架中。

核心创新

相比于其前身Fast R-CNN，Faster R-CNN的主要创新在于：

  1. 区域提议网络（RPN）：完全替代了传统的选择性搜索（Selective Search）方法，通过深度学习自动生成高质量的候选区域，显著提高了区域提议的生成速度
  2. 特征共享机制：RPN和后续的检测网络共享相同的卷积特征，避免了重复计算，大幅减少了计算开销
  3. 端到端训练：整个网络可以联合训练，使得RPN和检测网络能够协同优化，获得更好的整体性能

算法流程

Faster R-CNN的整体流程可以分为四个主要阶段：

  1. 特征提取：使用卷积神经网络（如VGG16、ResNet）提取输入图像的特征图
  2. 区域提议：通过RPN在特征图上生成候选区域（proposals）
  3. 区域对齐：使用ROI Pooling将不同大小的候选区域对齐到固定大小的特征
  4. 分类与回归：对每个候选区域进行分类（判断物体类别）和边界框回归（精确定位）

数据集准备

PASCAL VOC 2007

我选择了PASCAL VOC 2007数据集进行实验，该数据集包含20个常见物体类别，共计9963张标注图像。在实际实现中，我们还需要添加一个背景类，因此总共有21个类别。

数据预处理

数据预处理是目标检测任务中的关键步骤，主要包括：

  1. 图像缩放：将图像的短边缩放到600像素，长边按比例缩放，保持原始宽高比
  2. 边界框调整：图像缩放后，对应的边界框坐标也需要同步调整
  3. 数据增强：为了增加模型的鲁棒性，我实现了以下数据增强策略：
    • 随机水平翻转（概率0.5）
    • 颜色抖动（亮度、对比度、饱和度、色调的微小调整）
  4. 标准化：将像素值从[0, 255]归一化到[0, 1]，并进行标准化处理

class VOCDataset(Dataset):
    def __init__(self, root_dir, transform=None, is_train=True):
        self.root_dir = root_dir
        self.transform = transform
        self.is_train = is_train
        
        # 加载图像和标注文件路径
        self.image_paths = []
        self.annotation_paths = []
        
        # 这里省略了具体的文件加载逻辑
        # ...
    
    def __len__(self):
        return len(self.image_paths)
    
    def __getitem__(self, idx):
        # 加载图像
        image = cv2.imread(self.image_paths[idx])
        image = cv2.cvtColor(image, cv2.COLOR_BGR2RGB)
        
        # 解析XML标注文件
        boxes, labels = self.parse_annotation(self.annotation_paths[idx])
        
        # 应用数据增强
        if self.transform:
            transformed = self.transform(image=image, bboxes=boxes, class_labels=labels)
            image = transformed['image']
            boxes = transformed['bboxes']
            labels = transformed['class_labels']
        
        # 转换为张量
        image = torch.from_numpy(image).permute(2, 0, 1).float() / 255.0
        
        # 构建目标字典
        target = {
            'boxes': torch.tensor(boxes, dtype=torch.float32),
            'labels': torch.tensor(labels, dtype=torch.int64)
        }
        
        return image, target
    
    def parse_annotation(self, xml_path):
        tree = ET.parse(xml_path)
        root = tree.getroot()
        
        boxes = []
        labels = []
        
        for obj in root.findall('object'):
            # 获取类别标签（VOC类别从1开始，0保留给背景）
            class_name = obj.find('name').text
            label = self.class_to_idx[class_name] + 1  # +1是因为0是背景类
            
            # 获取边界框坐标
            bndbox = obj.find('bndbox')
            xmin = float(bndbox.find('xmin').text)
            ymin = float(bndbox.find('ymin').text)
            xmax = float(bndbox.find('xmax').text)
            ymax = float(bndbox.find('ymax').text)
            
            boxes.append([xmin, ymin, xmax, ymax])
            labels.append(label)
        
        return boxes, labels

class AnchorGenerator(nn.Module):
    def __init__(self, base_size=16, scales=[8, 16, 32], ratios=[0.5, 1, 2]):
        super(AnchorGenerator, self).__init__()
        self.base_size = base_size
        self.scales = scales
        self.ratios = ratios
        
    def generate_base_anchors(self):
        """生成基础锚框"""
        base_anchors = []
        for scale in self.scales:
            for ratio in self.ratios:
                # 计算锚框的高度和宽度
                h = self.base_size * scale * np.sqrt(ratio)
                w = self.base_size * scale / np.sqrt(ratio)
                
                # 锚框以(0, 0)为中心
                base_anchors.append([-w/2, -h/2, w/2, h/2])
        
        return torch.tensor(base_anchors, dtype=torch.float32)
    
    def forward(self, feature_map):
        """
        为特征图的每个位置生成锚框
        
        输入: feature_map形状为(batch_size, 512, H, W)
        输出: anchors形状为(batch_size, H*W*9, 4)
        """
        batch_size, _, height, width = feature_map.shape
        
        # 生成基础锚框
        base_anchors = self.generate_base_anchors()  # (9, 4)
        
        # 生成网格偏移
        shift_x = torch.arange(0, width) * self.base_size
        shift_y = torch.arange(0, height) * self.base_size
        shift_y, shift_x = torch.meshgrid(shift_y, shift_x, indexing='ij')
        shifts = torch.stack([shift_x.ravel(), shift_y.ravel(),
                             shift_x.ravel(), shift_y.ravel()], dim=1)  # (H*W, 4)
        
        # 将基础锚框应用到所有位置
        anchors = base_anchors.view(1, 9, 4) + shifts.view(height*width, 1, 4)
        anchors = anchors.view(height*width*9, 4)
        
        # 扩展到batch维度
        anchors = anchors.unsqueeze(0).repeat(batch_size, 1, 1)
        
        return anchors  # (batch_size, H*W*9, 4)

class RPN(nn.Module):
    def __init__(self, in_channels=512, mid_channels=512, num_anchors=9):
        super(RPN, self).__init__()
        self.num_anchors = num_anchors
        
        # 3x3卷积层，用于特征提取
        self.conv = nn.Conv2d(in_channels, mid_channels, kernel_size=3, stride=1, padding=1)
        
        # 分类分支：每个锚框预测前景/背景
        self.cls_layer = nn.Conv2d(mid_channels, num_anchors * 2, kernel_size=1, stride=1, padding=0)
        
        # 回归分支：每个锚框预测边界框偏移
        self.reg_layer = nn.Conv2d(mid_channels, num_anchors * 4, kernel_size=1, stride=1, padding=0)
        
        # 锚框生成器
        self.anchor_generator = AnchorGenerator()
        
        # 提议生成器
        self.proposal_generator = ProposalGenerator()
    
    def forward(self, x):
        """
        前向传播
        
        输入: (batch_size, 512, H/16, W/16)
        输出: cls_logits, proposals
        """
        # 特征提取
        x = F.relu(self.conv(x))  # (batch_size, 512, H/16, W/16)
        
        # 分类分支
        cls_logits = self.cls_layer(x)  # (batch_size, num_anchors*2, H/16, W/16)
        cls_logits = cls_logits.permute(0, 2, 3, 1).contiguous()  # (batch_size, H/16, W/16, num_anchors*2)
        cls_logits = cls_logits.view(cls_logits.size(0), -1, 2)  # (batch_size, H/16*W/16*num_anchors, 2)
        
        # 回归分支
        reg_pred = self.reg_layer(x)  # (batch_size, num_anchors*4, H/16, W/16)
        reg_pred = reg_pred.permute(0, 2, 3, 1).contiguous()  # (batch_size, H/16, W/16, num_anchors*4)
        reg_pred = reg_pred.view(reg_pred.size(0), -1, 4)  # (batch_size, H/16*W/16*num_anchors, 4)
        
        # 生成锚框
        anchors = self.anchor_generator(x)  # (batch_size, H/16*W/16*num_anchors, 4)
        
        # 生成提议
        proposals = self.proposal_generator(anchors, cls_logits, reg_pred)
        
        return cls_logits, proposals

class ProposalGenerator(nn.Module):
    def __init__(self, pre_nms_top_n=12000, post_nms_top_n=2000, nms_thresh=0.7, min_size=16):
        super(ProposalGenerator, self).__init__()
        self.pre_nms_top_n = pre_nms_top_n
        self.post_nms_top_n = post_nms_top_n
        self.nms_thresh = nms_thresh
        self.min_size = min_size
    
    def forward(self, anchors, cls_logits, reg_pred):
        """
        生成区域提议
        
        输入:
            anchors: (batch_size, num_anchors, 4)
            cls_logits: (batch_size, num_anchors, 2)
            reg_pred: (batch_size, num_anchors, 4)
        
        输出: (batch_size, N, 4)，其中N <= post_nms_top_n
        """
        batch_size = anchors.size(0)
        
        # 获取前景分数
        fg_scores = F.softmax(cls_logits, dim=2)[:, :, 1]  # (batch_size, num_anchors)
        
        # 调整锚框位置
        proposals = self.apply_regression(anchors, reg_pred)
        
        # 裁剪提议到图像边界内
        proposals = self.clip_boxes(proposals)
        
        # 移除太小的提议
        keep = self.remove_small_boxes(proposals)
        proposals = proposals[keep]
        fg_scores = fg_scores[keep]
        
        # 按分数排序，选择前pre_nms_top_n个
        if proposals.size(0) > self.pre_nms_top_n:
            topk_scores, topk_indices = torch.topk(fg_scores, self.pre_nms_top_n)
            proposals = proposals[topk_indices]
            fg_scores = fg_scores[topk_indices]
        
        # 应用非极大值抑制
        keep = torchvision.ops.nms(proposals, fg_scores, self.nms_thresh)
        
        # 选择前post_nms_top_n个
        if keep.size(0) > self.post_nms_top_n:
            keep = keep[:self.post_nms_top_n]
        
        proposals = proposals[keep]
        
        return proposals.unsqueeze(0)  # 添加batch维度
    
    def apply_regression(self, anchors, reg_pred):
        """应用回归偏移到锚框"""
        # 解码回归参数
        dx = reg_pred[:, :, 0]
        dy = reg_pred[:, :, 1]
        dw = reg_pred[:, :, 2]
        dh = reg_pred[:, :, 3]
        
        # 计算锚框的中心点和宽高
        anchor_x = (anchors[:, :, 0] + anchors[:, :, 2]) / 2
        anchor_y = (anchors[:, :, 1] + anchors[:, :, 3]) / 2
        anchor_w = anchors[:, :, 2] - anchors[:, :, 0]
        anchor_h = anchors[:, :, 3] - anchors[:, :, 1]
        
        # 应用回归变换
        pred_x = anchor_x + dx * anchor_w
        pred_y = anchor_y + dy * anchor_h
        pred_w = anchor_w * torch.exp(dw)
        pred_h = anchor_h * torch.exp(dh)
        
        # 转换回边界框格式
        proposals = torch.stack([
            pred_x - pred_w / 2,
            pred_y - pred_h / 2,
            pred_x + pred_w / 2,
            pred_y + pred_h / 2
        ], dim=2)
        
        return proposals

class ROIPooling(nn.Module):
    def __init__(self, output_size=7):
        super(ROIPooling, self).__init__()
        self.output_size = output_size
    
    def forward(self, features, rois):
        """
        ROI Pooling操作
        
        输入:
            features: (batch_size, C, H, W) 特征图
            rois: (batch_size, num_rois, 4) 区域提议
        
        输出: (batch_size * num_rois, C, output_size, output_size)
        """
        batch_size, channels, height, width = features.shape
        num_rois = rois.size(1)
        
        # 这里简化实现，实际应该使用torchvision.ops.roi_pool
        output = []
        
        for b in range(batch_size):
            for i in range(num_rois):
                x1, y1, x2, y2 = rois[b, i]
                
                # 将坐标映射到特征图空间
                x1 = max(0, int(x1 / 16))
                y1 = max(0, int(y1 / 16))
                x2 = min(width - 1, int(x2 / 16))
                y2 = min(height - 1, int(y2 / 16))
                
                if x2 <= x1 or y2 <= y1:
                    # 无效的ROI，用零填充
                    roi_feature = torch.zeros(channels, self.output_size, self.output_size)
                else:
                    # 提取ROI区域
                    roi = features[b, :, y1:y2+1, x1:x2+1]
                    
                    # 自适应池化到固定大小
                    roi_feature = F.adaptive_max_pool2d(roi, self.output_size)
                
                output.append(roi_feature)
        
        return torch.stack(output) if output else torch.zeros(0, channels, self.output_size, self.output_size)

def rpn_cls_loss(cls_logits, gt_labels):
    """
    RPN分类损失
    
    参数:
        cls_logits: (batch_size, num_anchors, 2) 分类logits
        gt_labels: (batch_size, num_anchors) 真实标签（1=前景，0=背景，-1=忽略）
    
    返回: 分类损失
    """
    # 只计算有效标签的损失（忽略标签为-1的锚框）
    valid_mask = gt_labels >= 0
    valid_logits = cls_logits[valid_mask]
    valid_labels = gt_labels[valid_mask].long()
    
    if valid_logits.numel() == 0:
        return torch.tensor(0.0, device=cls_logits.device)
    
    return F.cross_entropy(valid_logits, valid_labels)

def smooth_l1_loss(pred, target, beta=1.0):
    """
    Smooth L1损失函数
    
    参数:
        pred: 预测值
        target: 目标值
        beta: 控制平滑区域的参数
    
    返回: Smooth L1损失
    """
    diff = torch.abs(pred - target)
    loss = torch.where(diff < beta, 0.5 * diff ** 2 / beta, diff - 0.5 * beta)
    return loss.mean()

def rpn_reg_loss(reg_pred, gt_boxes, gt_labels):
    """
    RPN回归损失
    
    参数:
        reg_pred: (batch_size, num_anchors, 4) 回归预测
        gt_boxes: (batch_size, num_anchors, 4) 真实边界框
        gt_labels: (batch_size, num_anchors) 真实标签
    
    返回: 回归损失
    """
    # 只计算前景锚框的回归损失
    fg_mask = gt_labels == 1
    if not fg_mask.any():
        return torch.tensor(0.0, device=reg_pred.device)
    
    fg_reg_pred = reg_pred[fg_mask]
    fg_gt_boxes = gt_boxes[fg_mask]
    
    return smooth_l1_loss(fg_reg_pred, fg_gt_boxes)

def rpn_total_loss(cls_logits, reg_pred, gt_labels, gt_boxes, lambda_reg=1.0):
    """
    RPN总损失
    
    参数:
        cls_logits: 分类logits
        reg_pred: 回归预测
        gt_labels: 真实标签
        gt_boxes: 真实边界框
        lambda_reg: 回归损失的权重
    
    返回: 总损失
    """
    cls_loss = rpn_cls_loss(cls_logits, gt_labels)
    reg_loss = rpn_reg_loss(reg_pred, gt_boxes, gt_labels)
    
    return cls_loss + lambda_reg * reg_loss

import torch.optim as optim
from torch.utils.data import DataLoader
import torchvision.ops

def train_model(model, train_loader, val_loader, num_epochs=120):
    """
    训练Faster R-CNN模型
    
    参数:
        model: Faster R-CNN模型
        train_loader: 训练数据加载器
        val_loader: 验证数据加载器
        num_epochs: 训练轮数
    """
    # 优化器
    optimizer = optim.SGD(
        model.parameters(),
        lr=0.001,
        momentum=0.9,
        weight_decay=0.0005
    )
    
    # 学习率调度器
    scheduler = optim.lr_scheduler.StepLR(optimizer, step_size=30, gamma=0.1)
    
    # 训练循环
    for epoch in range(num_epochs):
        model.train()
        train_loss = 0.0
        
        for batch_idx, (images, targets) in enumerate(train_loader):
            # 将数据移动到GPU
            images = images.cuda()
            
            # 前向传播
            cls_logits, proposals, detections = model(images)
            
            # 计算损失
            loss = compute_loss(cls_logits, proposals, detections, targets)
            
            # 反向传播
            optimizer.zero_grad()
            loss.backward()
            optimizer.step()
            
            train_loss += loss.item()
            
            if batch_idx % 50 == 0:
                print(f'Epoch {epoch}, Batch {batch_idx}, Loss: {loss.item():.4f}')
        
        # 更新学习率
        scheduler.step()
        
        # 验证
        val_loss = validate_model(model, val_loader)
        
        print(f'Epoch {epoch} completed. Train Loss: {train_loss/len(train_loader):.4f}, '
              f'Val Loss: {val_loss:.4f}')
        
        # 保存检查点
        if epoch % 10 == 0:
            torch.save({
                'epoch': epoch,
                'model_state_dict': model.state_dict(),
                'optimizer_state_dict': optimizer.state_dict(),
                'loss': train_loss/len(train_loader),
            }, f'checkpoint_epoch_{epoch}.pth')
